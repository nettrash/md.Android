# Welcome to md

**md** is a fast, private Markdown editor. Your documents are plain `.md`
files, everything stays on your device, and what you see in **Preview** is
exactly what you wrote in **Edit**.

## Three ways to look at a document

- **Edit** — the raw Markdown source, ready for typing.
- **Preview** — the rendered result, like this page.
- **Split** — both side by side, updating as you type.

Switch between them any time from the view-mode control.

## A taste of Markdown

You write plain text with a few light marks — **bold**, *italic*,
~~strikethrough~~, `inline code`, and [links](https://nettrash.me) — and md
turns it into a clean, readable page.

> Blockquotes set a passage apart, like this one.

## Explore the other examples

- **Formatting** — headings, emphasis, lists, quotes, and links.
- **Tables** — GitHub-style tables with column alignment.
- **Code** — inline code and fenced blocks with language hints.
- **Images** — remote, linked, and embedded images.
- **Math** — LaTeX formulas, typeset offline with KaTeX.
- **Diagrams** — Mermaid and PlantUML diagrams, drawn on-device.
- **Writer Tools** — page breaks, private notes, and in-document links.
- **Example Book** — a folder of articles that md reads as one book.

Happy writing!
