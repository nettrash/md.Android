# Preface

This folder is a **book**. In md, a folder of Markdown files is more than
a folder: the app reads it as one continuous work.

- Each `.md` file is an **article**.
- Each subfolder is a **chapter** that groups its articles.
- Numeric filename prefixes such as `01-` and `02-` set the reading
  order; names without a prefix follow alphabetically.

This particular book was unpacked from md's built-in Examples, and it is
now yours — rename, reorder, rewrite, or delete anything in it. The two
chapters that follow form a small manual for the app itself:

1. **Getting Started** — the editor, saving, and export.
2. **Going Further** — rich content and writing books of your own.

Turn the page.
