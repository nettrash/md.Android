# Writing a Book

You are reading one. A book in md is nothing more than a folder with a
little discipline:

```
My Book/
  01-Preface.md
  02-First Chapter/
    01-An Article.md
    02-Another Article.md
  03-Second Chapter/
    01-More.md
```

- The folder is the **book**, subfolders are **chapters**, and `.md`
  files are **articles**.
- Numeric prefixes (`01-`, `02-`, …) set the order; names without a
  prefix sort alphabetically after them.
- To reorder, just rename; to add a chapter, just add a folder.

> Because a book is only files and folders, it syncs, backs up, and
> version-controls like anything else you own.

Start your own: create a folder, add a `01-` file, and write.
