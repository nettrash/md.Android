# Formatting

Markdown keeps your hands on the keyboard: a few plain-text marks cover
almost everything a document needs. Compare this file in Edit and Preview.

## Headings

A `#` starts a heading — one per level, from `#` down to `######`.

### Like this third-level heading

Headings also power the **Contents** menu, so a well-structured document
is easy to navigate.

## Emphasis

Wrap text to style it: **bold**, *italic*, ***both at once***,
~~strikethrough~~, and `inline code` for filenames or commands.

## Links

- A plain link: [nettrash.me](https://nettrash.me)
- A link with a title: [the md website](https://nettrash.me "Shown on hover")
- An autolink: <https://nettrash.me>

## Lists

1. Ordered lists count for you.
2. They can nest:
   - a bulleted sub-point,
   - and another.
3. And task lists track progress:
   - [x] learn the basics
   - [ ] write something great

## Blockquotes

> Quotes set a passage apart — and they nest:
>
> > like a reply inside a reply.
